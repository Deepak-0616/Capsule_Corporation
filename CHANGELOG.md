# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-01

### Added
- Initial release of Capsule Corporation
- **Dashboard**: Main hub with 7 research modules
- **Tech Vault**: Dragon Radar, Hoi-Poi Capsule, Saiyan Scouter
- **Battle History**: 12 epic battles with timeline layout
- **Ki Science**: 10 Super Saiyan transformations
- **Character Studio**: 13+ character profiles with stats
- **Universe Lore**: All 12 universes with gods and angels
- **Entertainment Zone**: 6 fun facts and trivia
- **Toriyama Tribute**: Dedication to Akira Toriyama
- Light/Dark theme toggle
- Hamburger navigation menu
- Fully responsive design
- Smooth animations and transitions
- Custom Capsule Corporation favicon
- Automatic scroll-to-top on page navigation

### Features
- Authentic anime/manga screenshots throughout
- Dynamic timeline layout for battles
- Accordion-style character studio
- Sequential grid layout for tech items
- Interactive hover effects
- Mobile-optimized responsive design
- Accessibility features (data-testid attributes)

### Tech Stack
- React 19 with TypeScript
- Tailwind CSS for styling
- Framer Motion for animations
- Vite as build tool
- Radix UI components
- Wouter for routing
- Lucide React for icons

### Breaking Changes
- Clearly mark as BREAKING CHANGE
- Update documentation
